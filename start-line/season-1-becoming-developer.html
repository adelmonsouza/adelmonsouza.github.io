Mock Interview Drill (Episode 5)

  • Q: What problem does encapsulation solve?
    A: It hides internal state, preventing unintended changes and making the class easier to reason about; consumers interact through well-defined methods.
  • Q: When would you use an enum?

  • Q: How do you decide when to create a new class or keep logic in an existing one?
    A: If the logic represents a distinct responsibility, changes independently, or makes testing easier when isolated, I extract it into a new class.
